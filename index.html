Introduction

Chan's algorithm partitions the set of \(n\) points \(P\) into \(\Bigl\lceil \dfrac{n}{m} \Bigr\rceil\) groups of size at most \(m\) each. For each group \(P_1, P_2, ..., P_{\lceil n/m \rceil}\), we compute the convex hull. The figure below is an example with the outcome after this first step, i.e., we end up with four subsets of \(P\) each with \(\leq m\) points (and their individual hulls are constructed).

Overall, Chan's algorithm is divided into three stages:
  1. Parition \(P\) into \(\Bigl\lceil \dfrac{n}{m} \Bigr\rceil\) groups of size at most \(m\) each.
  2. Use Graham's Scan to find the convex hull of each group.
  3. Use Gift Wrapping to compute the convex hull of \(P\).
